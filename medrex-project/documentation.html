Useful Tips

*Due to local security restrictions, some things will not work locally, especially in Chrome for example Google maps will not work until you will not activate API key.
*If you encounter an error Failed to execute 'postMessage' on 'DOMWindow': The target origin provided ('https://www.youtube.com') does not match the recipient window's origin, you need an SSL certificate for your website, all youtube video will work the same as before even if your not having one installed.
Html Structure

Html structure of this template is divided by sections, you will find header, main, and footer area. You can choose between headers in case if you need map or usual info header. Build your page from existing code blocks. Each block is separated and has his own styles. Buttons, texts are all standard and were used multiple times.

In order to disable ease effect, on page scroll, just remove a div with mt-momentum class and the page will act like a regular one with normal scrolling.

Each element outside "mt-momentum" has position fixed. Please note that position fixed does not work inside "mt-momentum" div because page actually is not scrollable, so all fixed elements remain in the same position, even if the page is scrolled.

In order to make navbar static and not fixed just remove "mt-t1-navbar--fixed" class and place all navbar code inside of the header tag.

You can control your page layout with entire bootstrap 4 style library, all components you can find here "https://getbootstrap.com/docs/4.0/components/buttons/" as well you can adjust paddings and margins between elements by playing with additional classes like:

pt-10 pr-10 pb-10 pl-10 - will set padding 10 on each side of selected element.

Same structure for margins:

mt-10 mt-10 mt-10 mt-10 - will set margin 10 on each side of selected element.

Each class on resolution lower than 1200px will be converted to half of it`s value for example if you used mt-50 it will set 25px margin top, on screens less than 1200px and so on for the other classes. Exception are only classes with value less than 20px. Space amounts used in this template (0, 5, 10, 15, 20, 30, 40, 50, 60, 80, 90, 100, 250), you can modify scss mixin called spaces inside of __general.scss file and recompile it with sass/scss compiler.

Each section can be easily animated by adding "animated" class, please also specify animation type required inside animation attribute. For example see this code structure below:

<div class="mt-t1-benefits animated" data-animation="fadeInRight"></div>

for animation was used animate.css stylesheet. https://daneden.github.io/animate.css/, you can find all animations on developers website.

Each element on the page can be scroll trigger or scroll target, for example:

<div class="mt-t1-benefits" data-scroll-aim='my-form'></div>

Will scroll the page to:

<div class="mt-t1-form" data-scroll-target='my-form'></div>

as you can see above, element with data-scroll-aim attribute is the trigger in these case, and the element with a data-scroll target attribute is the scroll point. You can place as much scroll triggers as you want.

In such way you can set menu links, buttons a.o elements to scroll to any selected element target on the page.

All tempale svg icons were converted to icon-font, using fontello service, http://fontello.com/.

Each icon contain mt-icon class, all used icons can be found by accesing demo.html file inside of fontello-aa311037 folder. All svg files are maintained as well, if you would like to add new icons to the set, you can upload font inside fontello-aa311037 folder to fontello website in order to modify it. Please be careful while changing font structure, make a copy before doing that.


Scss/Css Structure

In order to edit page css please use scss to css converter like Koala or Scout application for example. If you would like to use clean css whithout preprocessors use this https://unminify.com/ or any other related service to unminify css code. Styles are separated and all are imported into main.scss file. Styles are written mostly using BEM methodology.

__animate.scss contain all animations from Animate.css library.
__elements.scss styles for each section.
__footer.scss styles for footer.
__general.scss mixins, variables, and general style rules.
__headers.scss styles for all page headers are here.
__helpers.scss additional classes for small elements on the page like buttons, text a.o.
__navigation.scss styles for navigation bar.
__plugins.scss contain styles modifications for included plugins and G-maps API.

Media Queries are included into each class, for example you can include media mixin in such way: 

                    
                        .mt-t1-element {
                            &__block {
                               "Css rules before breakpoint"
                            }
                            @include max-media($break-xs) {
                                &__block {
                                    "Css rules after breakpoint"
                                }
                            }
                        }

As well you can set min-media: 

                    
                        .mt-t1-element {
                            &__block {
                               "Css rules before breakpoint"
                            }
                            @include min-media($break-xs) {
                                &__block {
                                    "Css rules after breakpoint"
                                }
                            }
                        }

Youtube API

                        
var tag = document.createElement('script');
tag.src = "http://www.youtube.com/player_api";
var firstScriptTag = document.getElementsByTagName('script')[0];
firstScriptTag.parentNode.insertBefore(tag, firstScriptTag);

var player;
var page_url = window.location.href;
var page_url = String(page_url.replace("http", "https"));

function onYouTubePlayerAPIReady() {
    player = new YT.Player('mt-t2-video__box', {    <<--- class for Youtube embed.
        height: '100%',
        width: '100%',
        playerVars: {            <<--- You can set youtube API options here.
            'controls': 0, 
            'autohide': 1,
            'showinfo' : 0,
            'wmode': 'opaque',
            'rel': 0,
            'modestbranding': 1
        },
        host: page_url,
        videoId: 'XKu_SEDAykw'     <<--- video id, you can find it at the end of any video url.
    });
}
Google Maps API

                        
function CoordMapType(tileSize) {
    this.tileSize = tileSize;
}

CoordMapType.prototype.getTile = function(coord, zoom, ownerDocument) { 
    var div = ownerDocument.createElement('span');
    div.style.display = 'block';
    div.style.width = this.tileSize.width + 'px';
    div.style.height = this.tileSize.height + 'px';
    div.style.backgroundColor = '#000';                      <<--- map overlay color
    div.style.opacity = 0.5;                                 <<--- map opacity
    return div;
};

var map;

function initMap() {
    var myLatLng = {lat: 41.875671, lng: -87.634224};         <<--- marker coordinates

    var icon = {                                              <<--- icon settings
        url: '../img/icons/google-maps/marker.svg',
        scaledSize: new google.maps.Size(38, 53),
        origin: new google.maps.Point(0,0),
        anchor: new google.maps.Point(0, 0)
    };

    map = new google.maps.Map(document.getElementById('map'), {
        zoom: 16,                                              <<--- set map zoom here
        center: myLatLng
    });

    var marker = new google.maps.Marker({                      <<--- marker options
        position: myLatLng,
        map: map,
        icon: icon,
        title: 'Click to open'
    });

    var contentString =                                        <<--- Info window on marker click
        "<div class='mt-gmaps'>" +
        "<div class='mt-gmaps__street'>NE Killingsworth St 62</div>" +
        "<div class='mt-gmaps__city'>Hopewell, VA 23860</div>" +
        "</div>";

    var infowindow = new google.maps.InfoWindow({
        content: contentString
    });

    marker.addListener('click', function() {
        infowindow.open(map, marker);
    });

    map.overlayMapTypes.insertAt(0, new CoordMapType(new google.maps.Size(256, 256)));
}
